Machine Learning Breast Cancer Detection Web App

This code is done in python using Datature for creating a tensor flow model and using Kaggle for my dataset (citation below).

I created a machine learning/deep learning technique to automate identification and diagnosis of breast cancer using my computer vision tensorflow model. Using the a dataset from kaggle, I downloaded ultrasound images of normal, benign, and malignant cases. In order to train a model, I needed to have a four corner csv to map out where the benign and malignant detectable nodules were in the ultrasound. so using the masks included in the dataset, I had a python script to read in the mask files and map out the shape in the mask with the xy coordinates and height/width dimensions of the mask and write it to a csv.

Once I had the csv created, I could upload the dataset ultrasound images into datature and use my annotations.csv as my four corner annotation method to auto annotate my images. I then, created a workflow on datature to create my tensorflow model using these images and annotations.

Although my dataset had up to 800 images, my datature account allowed less than 800 so I decreased my image range losing some training.
